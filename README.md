# Controle Financeiro 🪙

Este é um projeto de gestão financeira pessoal desenvolvido para colocar em prática conceitos de **Programação Orientada a Objetos (POO)** e manipulação de dados em Python.

## 🚀 Funcionalidades
- **Cadastro de Movimentações:** Registro de receitas e despesas com categorização.
- **Gestão de Saldo:** Cálculo automático de saldo livre e reserva acumulada.
- **Filtros Temporais:** Visualização de resumos mensais e anuais.
- **Interface Gráfica:** Desenvolvida para facilitar o uso diário do sistema.

## 🛠️ Tecnologias Utilizadas
- **Python**: Linguagem principal.
- **Tkinter/CustomTkinter**: Para a interface do usuário (GUI).
- **SQLite/Pandas**: (ajuste conforme o que você usou para salvar os dados) para persistência e manipulação.

## 📊 Estrutura do Projeto
O sistema permite o acompanhamento de metas de reserva e o controle rigoroso de gastos totais, ajudando na saúde financeira e no planejamento de longo prazo.
